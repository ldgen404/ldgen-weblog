# ldgen-weblog

一个基于 `Spring Boot 3 + MyBatis-Flex + Sa-Token + Vue 3 + Vite + Element Plus` 的前后端分离博客项目。

这是我的个人博客项目，我会将自己在技术学习和实践过程中的思考持续沉淀到这里。

博客内容目前主要围绕以下几个方向展开：

- 大模型应用学习与实践
- Java 底层原理深入学习
- JVM、JUC 等核心知识整理
- 日常学习感想、总结与阶段性复盘

我希望通过这个博客，一边完整搭建属于自己的博客系统，一边持续输出有沉淀、有思考的技术内容。

当前仓库的定位不是简单的博客模板，而是一个持续迭代的全栈博客实战项目，覆盖后台管理、博客前台、文章体系、搜索体系，以及后续的知识库、评论、聊天室等功能。

## 项目目标

### 一期目标

一期主要完成一个可上线的前后端分离博客系统，包含以下核心能力：

1. 后端基础设施
- Spring Boot 工程搭建
- 多环境配置
- 日志打印与请求日志切面
- 参数校验、统一响应体、全局异常处理
- Knife4j 接口文档

2. 用户与登录体系
- 登录页
- Token 鉴权
- 登录态校验
- 获取当前登录用户
- 修改密码、退出登录

3. 后台管理系统
- Admin 布局骨架
- 顶部 Header、左侧 Menu、标签导航栏、Footer
- KeepAlive 缓存与页面切换动画
- 仪表盘

4. 内容管理
- 分类管理
- 标签管理
- 博客设置
- 文章发布、编辑、删除、列表管理

5. 博客前台
- 首页
- 分类页
- 标签页
- 归档页
- 文章详情页
- 前台 Header / Footer / 右侧边栏

6. 部署上线
- MySQL
- 对象存储
- Nginx 反向代理
- 前后端部署
- 域名与 HTTPS

### 二期目标

二期在一期基础上继续增强博客的内容能力和产品完整度，当前规划包含：

1. 站内搜索
- 博主信息卡片统计接口
- 文章、分类、标签、总访问量展示
- Lucene 全文检索
- 中文分词搜索
- 搜索结果高亮
- 搜索弹窗与移动端适配
- 文章新增、编辑、删除时同步索引

2. 前台体验增强
- 白天 / 黑夜主题切换
- 代码复制
- 页面交互和视觉细节优化

3. 工程化与交付
- CI / CD 持续集成与部署
- 更规范的发布流程

4. 内容社区能力
- 知识库后台
- 知识库前台
- 评论模块
- 公共聊天室

5. 风控与安全
- 行为验证码
- 滑块 / 点选等验证能力

## 本地启动

### 1. 启动后端

```bash
./mvnw spring-boot:run
```

默认配置：

- 端口：`8123`
- 上下文：`/api`

### 2. 启动前端

```bash
cd weblog-vue3
npm install
npm run dev
```

默认前端开发地址：

- `http://localhost:5173`

前端已配置代理：

- `/api -> http://localhost:8123`

## 开发说明

1. 本项目参考犬小哈专栏，但会结合当前仓库实际情况做适配，不完全拘泥于原专栏代码实现。
2. 当前仓库后端鉴权采用的是 `Sa-Token`，不是教程里常见的 `Spring Security + JWT` 组合。
3. 当前仓库 ORM 采用的是 `MyBatis-Flex`，不是 `MyBatis-Plus`。
4. 后续 README 会随着一期补全、二期推进持续更新。
